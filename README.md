# Chicago Taxi Trips Analysis

## Overview
Analysis of Chicago taxi trips (2023) using BigQuery public dataset.

## Stack
- BigQuery (public dataset)
- Python (pandas, matplotlib)
- Jupyter Notebook

## Key Findings
- Flash Cab leads the market with 19.39% share
- Peak hours: 15:00-18:00 (evening rush hour)
- Credit card users tip 22% vs 0.12% for cash

## How to Run
1. Install: `pip install google-cloud-bigquery pandas matplotlib`
2. Authenticate: `gcloud auth application-default login`
3. Run `analysis.ipynb`
